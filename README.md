
# 🎓 Прогнозирование успеваемости студентов

## Описание
Веб-приложение (дашборд) для предсказания статуса студента: отчисление (Dropout), 
продолжение обучения (Enrolled) или выпуск (Graduate).

Построено на основе датасета **"Predict Students' Dropout and Academic Success"** (UCI ML Repository).

## Используемые модели

| Модель | Тип | F1-score |
|--------|-----|----------|
| Stacking | Ансамбль (стэкинг) | 0.7191 |
| Gradient Boosting | Ансамбль (бустинг) | 0.7158 |
| Random Forest | Ансамбль (бэггинг) | 0.7117 |
| CatBoost | Продвинутый бустинг | 0.7037 |
| Neural Network | Нейронная сеть | 0.6966 |
| Logistic Regression | Классическая модель | 0.6797 |
